# Design Concepts

Эта папка предназначена для 5 альтернативных HTML-концептов нового визуального стиля.

Планируемые файлы:
- `concept-1.html`
- `concept-2.html`
- `concept-3.html`
- `concept-3-1.html` (Concept 3 + Liquid Glass плашки)
- `concept-4.html` (Liquid Glass)

Запуск локально:
1. В корне репозитория выполнить `python3 -m http.server 4173`
2. Открыть в браузере:
   - `http://127.0.0.1:4173/design-concepts/concept-1.html`
   - `http://127.0.0.1:4173/design-concepts/concept-2.html`
   - `http://127.0.0.1:4173/design-concepts/concept-3.html`
   - `http://127.0.0.1:4173/design-concepts/concept-3-1.html`
   - `http://127.0.0.1:4173/design-concepts/concept-4.html`

После согласования концепта выбранный стиль переносится в рабочие страницы приложения.
